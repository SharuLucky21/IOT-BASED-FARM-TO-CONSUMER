
# Flask + Firebase Backend

## Steps to Run

1. Install Python 3.8+
2. Install dependencies:
   pip install -r requirements.txt

3. Download Firebase service account key:
   Firebase Console → Project Settings → Service Accounts → Generate Key

4. Rename the key file to:
   firebase_key.json

5. Place firebase_key.json in this folder

6. Edit app.py and replace:
   YOUR_PROJECT_ID

7. Run:
   python app.py

Server URL:
http://127.0.0.1:5000/
